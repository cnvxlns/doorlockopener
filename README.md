🔑 Doorlock Opener

지문 인식을 통해 공동 현관문을 제어하는 스마트 도어락 오프너 앱입니다.
보안성과 편의성을 동시에 제공하여, 사용자는 별도의 키나 비밀번호 입력 없이 본인 인증 후 출입이 가능합니다.
